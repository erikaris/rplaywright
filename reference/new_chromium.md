# Launch new chromium instance

Launch new chromium instance

## Usage

``` r
new_chromium(
  start_server = T,
  serverOptions = list(host = "127.0.0.1", port = 3000)
)
```

## Value

An object of class `Browser`

## Examples

``` r
if (FALSE) { # \dontrun{
browser <- rplaywright::new_chromium()
} # }
```
