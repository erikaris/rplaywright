# Creates a new page in the browser context.

Creates a new page in the browser context.

## Usage

``` r
new_page(context)
```

## Arguments

- context:

  Browser Context instance

## Examples

``` r
if (FALSE) { # \dontrun{
 rplaywright::new_chromium() %>%
  new_context() %>%
  new_page()
} # }
```
